My work:

DDPERFORMANCE

Not so long ago I did a project for a course at Software University Bulgaria. The course in question was called CSS Advanced. The final project was to build a website using a static blog generator called Jekyll. A part of the assignment was to use SCSS, in combination with Node.js, and packages like Gulp, browser sync and css minifier.
In the end, I managed to get a deeper insight into how CSS and SASS work, along with building layouts using Flexbox and Grid. I also learned how to combine a separate program like Jekyll with Node.js and its packages.
